# Олимпиадная школа — клон-демо

Демонстрационный клон лендинга [kruzhok.school](https://kruzhok.school/), собранный на **Next.js 16 + React 19 + TypeScript + Tailwind CSS 4**.

> Структура, секции и общая стилистика повторяют оригинал, но тексты и иллюстрации заменены на нейтральные плейсхолдеры — авторский контент с оригинала не копировался.

## Стек

- **Next.js 16** (App Router, Turbopack)
- **React 19**
- **TypeScript 5**
- **Tailwind CSS 4** (PostCSS)
- **ESLint 9**

## Что внутри

Главная страница (`src/app/page.tsx`) собрана из компонентов:

- `Header` — закреплённый хедер с логотипом и навигацией
- `Hero` — заголовок с 7 цветными названиями предметов и инфо-карточкой
- `About` — раздел «Что такое школа» с миссией и плитками
- `Likbez` — блок про форматы олимпиад (ВсОШ, перечневые)
- `Courses` — сетка 7 курсов с цветовой кодировкой
- `AppSection` — три мокапа мобильного приложения
- `Stats` — карточки статистики (5 лет / 1000+ / −13% / БВИ)
- `Results` — карточки результатов и цветовая раскладка
- `Universities` — логотипы вузов и горизонтальная карусель выпускников
- `Reviews` — отзывы с marquee-лентой
- `CTA` — тёмная секция «Записаться на курс»
- `Footer`

Все данные вынесены в `src/data/site.ts`, цветовые переменные и анимации (`marquee`, фоновая сетка) — в `src/app/globals.css`.

## Запуск

Требуется Node.js 18.18+ (рекомендуется 20+).

```bash
# 1. Установить зависимости
npm install

# 2. Поднять dev-сервер
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

### Полезные команды

```bash
npm run dev      # dev-сервер (Turbopack)
npm run build    # production-сборка
npm run start    # запуск production-сборки
npm run lint     # eslint
```

## Структура

```
src/
├── app/
│   ├── globals.css     # цвета, marquee, сетка-фон
│   ├── layout.tsx      # корневой layout + шрифт Inter (cyrillic)
│   └── page.tsx        # главная страница, импортирует все секции
├── components/         # Header, Hero, About, Likbez, Courses, AppSection,
│                       # Stats, Results, Universities, Reviews, CTA, Footer,
│                       # Logo, SectionLabel
└── data/
    └── site.ts         # данные: предметы, отзывы, выпускники, вузы и т.д.
```

## Замена контента

Все тексты, отзывы, имена выпускников, статистика и описания курсов лежат в `src/data/site.ts` — можно подменить на свои данные, не залезая в компоненты.

Цвета направлений и фоновой сетки настраиваются CSS-переменными в `src/app/globals.css` (`--c-history`, `--c-society` и т.д.).

## Лицензия

Демонстрационный код для собственного использования. Не предназначен для коммерческого деплоя без замены оставшихся отсылок к оригиналу.
